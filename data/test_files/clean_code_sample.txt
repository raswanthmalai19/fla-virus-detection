Python Function Example:
def calculate_total(items):
    total = 0
    for item in items:
        total += item.price
    return total

JavaScript Function Example:
function validateEmail(email) {
    const regex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
    return regex.test(email);
}

SQL Query Example (Safe):

FROM users 
WHERE status = 'active' 
ORDER BY created_at DESC;
